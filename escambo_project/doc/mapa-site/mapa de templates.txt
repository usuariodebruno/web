###Templates Necessários para a Plataforma###

base - directorio - dividir arquivos de estrutura geral do site, incluindo o cabeçalho, rodapé e elementos comuns a todas as páginas.

index.html - Template para a página inicial do site.
login.html - Template para a página de login.
perfil_usuario.html - Template para exibir o perfil do usuário.
editar_perfil.html - Template para editar o perfil do usuário.
meus_produtos.html - Template para exibir os produtos do usuário.
publicar_produto.html - Template para publicar um novo produto.
editar_produto.html - Template para editar um produto existente.
excluir_produto.html - Template para confirmar a exclusão de um produto.
minhas_trocas.html - Template para exibir as trocas do usuário.
detalhes_troca.html - Template para exibir os detalhes de uma troca.
confirmar_recebimento.html - Template para confirmar o recebimento de um produto na troca.
avaliar_usuario.html - Template para avaliar um usuário após a conclusão da troca.
cadastro.html - Template para a página de cadastro de novos usuários.
de_cima.html - Template para exibir os produtos da Categoria 1.
para_enfeitar.html - Template para exibir os produtos da Categoria 2.
dos_pes.html - Template para exibir os produtos da Categoria 3.
de_baixo.html - Template para exibir os produtos da Categoria 4.
pesquisar_produtos.html - Template para a página de pesquisa de produtos.
detalhes_produto.html - Template para exibir os detalhes de um produto.
chat_escambo.html - Template para o chat de escambo entre os usuários.
